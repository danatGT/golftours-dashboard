How to include a new campaign

  1. Open the HTML file and find CAMPAIGN_FILTER near the top of the <script> block.
  2. Change the value to your new campaign name exactly as it appears in Meta Ads Manager, e.g. "GT - Traffic - 2027".
  3. To show multiple campaigns, change the operator to in and list names comma-separated: ["campaign","in","Campaign A, Campaign B"].
  4. To show all active campaigns, replace the filter entirely with ["campaign_status","eq","ACTIVE"].
  5. Save and redeploy — data will refresh automatically on every page load via Windsor.ai.
