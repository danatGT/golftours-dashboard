Adding campaigns

How to add a new campaign

  1. Open public/index.html on GitHub and find CAMPAIGN_FILTER in the script.
  2. Change the campaign name to match exactly what appears in Meta Ads Manager.
  3. Multiple campaigns: ["campaign","in","Campaign A, Campaign B"]
  4. All active: ["campaign_status","eq","ACTIVE"]
  5. Commit — Vercel redeploys automatically.
