Using this dashboard

  1. Use the toggle buttons above to switch between GT Enquiries 2026 and GT Brand Awareness 2026.
  2. Brand Awareness shows Reach, Frequency and CPM instead of CTR / CPC / Cost per Enquiry.
  3. Name updated creatives with v2, v3 etc. and they group automatically.
  4. Ad set names like GT - USA East Coast - Bay Hill display as-is in all tables.
  5. Name A/B variants with suffix Ad Name - A / Ad Name - B to surface them in section 07.
